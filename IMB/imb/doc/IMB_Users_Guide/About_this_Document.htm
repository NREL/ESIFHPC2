Intel® MPI Benchmarks User Guide and Methodology Description

About this Document

The objectives of the Intel® MPI Benchmarks User Guide are:

  • Provide a concise set of benchmarks targeted at measuring the most important MPI functions.

  • Set forth a precise benchmark methodology.

  • Report bare timings rather than provide interpretation of the measured results. Show throughput values if and only if these values are well-defined.

Intended Audience

This guide is intended for users who want to measure performance of MPI implementations.
